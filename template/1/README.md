- `set_section_titles.vba`
    - `SetSectionTitles`
        - 章タイトルを入力する.
            - 入力はテキストボックスにカンマ(,)で区切って入力する.
        - Agenda スライドが作成される.
            - 章が変わるときに、次の章を色付けして表示する.
            - 円の図形で現在の章を表す.
        - コンテンツスライドが作成される.
            - コンテンツスライドの "SectionTitle" に章タイトルを表示する.
            - 円の図形で現在の章を表す.
